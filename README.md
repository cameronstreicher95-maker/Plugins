# Plugins
Plug-in for Minecraft Java servers

## Download
Download the built plugin jar here:
- [spawnplugin-1.0.0.jar](target/spawnplugin-1.0.0.jar)

## Install on Paper
1. Place the jar file into your Paper server's plugins folder.
2. Start or restart the server.
3. Use /plsetspawn to set the spawn point.
4. Players can use /spawn.

## Commands
- /spawn - Teleport to the saved spawn point
- /plsetspawn - Set the spawn point (op only)

## Permissions
- plspawn.use - Allows players to use /spawn
- plspawn.admin - Allows staff to set the spawn point
